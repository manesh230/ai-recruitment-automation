# AI Recruitment Automation System

## What This Project Does
Automated recruitment system that processes resumes 
and scores candidates using AI.

## Workflow
1. Candidate uploads resume (PDF)
2. System saves to Google Drive
3. AI extracts candidate information
4. Data stored in Supabase database
5. AI scores candidate vs job description
6. Recruiter receives email report automatically

## Tech Stack
- n8n (Automation)
- OpenAI GPT-4o-mini (AI Processing)
- Supabase (Database)
- Google Drive (File Storage)
- Gmail (Email Reports)

## Results
- Match Score calculation
- Skill gap analysis
- Auto-generated interview questions
- Instant recruiter notification
